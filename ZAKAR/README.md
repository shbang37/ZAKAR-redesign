// ZAKAR

ZAKAR는 교회 사역 환경에서 촬영되는 많은 사진을 빠르게 정리하고, 유사 사진을 분석해 중복을 줄이며, 앨범/공유/백업을 돕는 iOS 앱입니다.

## 주요 기능
- 유사 사진 분석 및 그룹화
  - pHash + Hamming Distance 기반 유사도 분석
  - 시간 인접도(촬영 시각)와 시각적 유사도를 함께 반영
- 빠른 정리 워크플로
  - CleanUpView에서 제스처로 빠르게 선별
  - 위(삭제 후보), 아래(즐겨찾기), 좌/우(이동)
  - 임시보관함에서 선택/전체 복구 및 삭제
- 앨범 관리/공유
  - 새 앨범 생성, 앨범에 추가
  - 공유(파일명 규칙 적용) 지원
- 온보딩/권한 안내
  - 첫 실행 온보딩, 사진 권한 요청, 튜토리얼
- 드라이브 연동(구글 드라이브, 베타)
  - OAuth(PKCE) 기반 인증 뼈대
  - 업로드/동기화 API 설계

## 빌드 요구 사항
- Xcode 15+ (권장: Xcode 26)
- iOS 17+ 타겟 (iOS 16에서도 동작 가능하나 NavigationStack 권장)
- Swift 5.9+

## 프로젝트 구조(요약)
- ZAKARApp.swift: 엔트리 포인트, `PhotoManager` 주입 및 초기 로드
- MainTabView.swift: 홈/앨범/업로드 탭 구성, 온보딩 표시
- HomeView.swift: 홈 화면(UI), 요약/네비게이션, NavigationStack 기반
- ContentView.swift: 유사 사진/모든 사진 화면, 세그먼트 전환, 정리 흐름
- Components.swift: 재사용 컴포넌트(썸네일, 휴지통 버튼 등)
- PhotoManager.swift: 사진 로드/유사 분석/앨범 관리 핵심 로직
- CleanUpView.swift: 제스처 기반 빠른 선별 화면
- LocalDB.swift: 메타데이터/임시보관함(영속) 저장소
- GoogleDriveService.swift: 구글 드라이브 연동(베타)

자세한 아키텍처 설명은 ARCHITECTURE.md를 참고하세요.

## 권한
- 사진 접근(Photos): 유사 사진 분석/정리/앨범 추가/공유에 필요

## 첫 실행 흐름
1) 온보딩 → 권한 요청 → 튜토리얼
2) 홈 화면 → 유사 사진/모든 사진 진입
3) 유사 사진 탭 선택 시 자동 분석(필요 시 지연 분석 큐로 보장)

## 임시보관함(영속)
- `LocalDB.trash.json`에 PHAsset.localIdentifier 목록을 저장/복원
- 앱 재시작 후에도 삭제 후보 유지

## 드라이브 연동(베타)
- Google OAuth(PKCE) 뼈대 구현
- 실제 clientID/redirectScheme 설정 필요
- 업로드/폴더 생성/경로 보장 로직 포함

## 실행/테스트 팁
- 첫 실행 시 온보딩이 뜨지 않으면 앱 삭제 후 재설치
- 홈 → 유사 사진 블록을 반복 탭해도 항상 유사 탭으로 진입해야 함
- 유사 그룹 수는 홈 요약 카드에 반영됨

## 로드맵(요약)
